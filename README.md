# FiguriSwap 2026

## Descripción del proyecto

FiguriSwap 2026 es una aplicación móvil desarrollada como parte de la asignatura Diseño de Software de Dispositivos Móviles de la Tecnicatura Universitaria en Diseño de Software (FTyCA - UNCA).

La aplicación busca facilitar el intercambio seguro de figuritas del Mundial 2026 mediante un sistema de coincidencias entre usuarios y el uso de kioscos adheridos como puntos de intercambio confiables.

Esta primera versión implementa una porción vertical del producto que permite validar las funcionalidades principales del sistema.

---

## Problema que resuelve

Los coleccionistas de figuritas suelen utilizar grupos de WhatsApp, Facebook o encuentros presenciales para realizar intercambios. Esto genera diversos problemas:

* Riesgo de estafas o incumplimiento de acuerdos.
* Dificultad para coordinar horarios y lugares de encuentro.
* Exposición de datos personales.
* Desorganización en la búsqueda de figuritas faltantes.
* Pérdida de tiempo al intentar encontrar coincidencias compatibles.

FiguriSwap propone una alternativa más segura y organizada mediante acuerdos digitales y puntos de intercambio verificados.

## Limitaciones de esta versión

Esta versión corresponde a un MVP académico y presenta las siguientes limitaciones:

* El sistema de kioscos adheridos se encuentra simulado.
* No se implementó procesamiento de pagos.
* No se implementó chat entre usuarios.
* No se implementó geolocalización en tiempo real.
* Algunas funcionalidades avanzadas fueron planificadas para versiones futuras.
